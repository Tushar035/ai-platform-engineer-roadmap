📌 1. Getting Started with FastAPI

✅ Installation

pip install fastapi uvicorn

✅ Basic App

from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def home():
    return {"message": "Hello FastAPI 🚀"}

▶️ Run Server

uvicorn main:app --reload

main → file name

app → FastAPI instance


🌐 Access

API: http://127.0.0.1:8000

Swagger UI: http://127.0.0.1:8000/docs

ReDoc: http://127.0.0.1:8000/redoc

📌 2. Path Parameters Deep Dive

Path parameters are values embedded in the URL.

✅ Basic Example

@app.get("/items/{item_id}")
def get_item(item_id: int):
    return {"item_id": item_id}

👉 URL:

/items/10


---

🔍 Type Validation

FastAPI automatically validates types:

@app.get("/users/{user_id}")
def get_user(user_id: int):
    return {"user_id": user_id}

❌ /users/abc → Error
✅ /users/1 → Works


---

🎯 Multiple Path Parameters

@app.get("/users/{user_id}/orders/{order_id}")
def get_order(user_id: int, order_id: int):
    return {
        "user_id": user_id,
        "order_id": order_id
    }


---

🧠 Enum Path Parameters

from enum import Enum

class Category(str, Enum):
    books = "books"
    electronics = "electronics"

@app.get("/products/{category}")
def get_products(category: Category):
    return {"category": category}


---

⚠️ Path Order Matters

@app.get("/items/me")
def read_me():
    return {"user": "me"}

@app.get("/items/{item_id}")
def read_item(item_id: str):
    return {"item_id": item_id}

✔️ /items/me must come first


---

🎯 Path Parameter Metadata

from fastapi import Path

@app.get("/items/{item_id}")
def get_item(item_id: int = Path(..., title="Item ID", ge=1)):
    return {"item_id": item_id}

Useful Options:

ge → greater than or equal

le → less than or equal

title, description

📌 3. Query Parameters Mastery

Query parameters come after ? in URL.

✅ Basic Example

@app.get("/items/")
def get_items(skip: int = 0, limit: int = 10):
    return {"skip": skip, "limit": limit}

👉 /items/?skip=5&limit=20


---

🔍 Optional Parameters

from typing import Optional

@app.get("/users/")
def get_users(name: Optional[str] = None):
    return {"name": name}


---

🎯 Required Query Parameter

from fastapi import Query

@app.get("/search/")
def search(q: str = Query(...)):
    return {"query": q}


---

📊 Validation in Query Params

@app.get("/products/")
def get_products(
    price: float = Query(..., gt=0),
    rating: int = Query(ge=1, le=5)
):
    return {"price": price, "rating": rating}


---

🔁 Multiple Values (List)

from typing import List

@app.get("/tags/")
def get_tags(tags: List[str]):
    return {"tags": tags}

👉 /tags/?tags=python&tags=fastapi


---

🧠 Boolean Query

@app.get("/items/")
def get_items(active: bool = False):
    return {"active": active}

👉 /items/?active=true


---

🎯 Query Metadata

@app.get("/items/")
def get_items(
    q: str = Query(None, min_length=3, max_length=10)
):
    return {"q": q}

📌 4. Request Body & Validation Basics

Used when sending data via POST/PUT.


---

✅ Basic Request Body

from pydantic import BaseModel

class Item(BaseModel):
    name: str
    price: float
    is_available: bool

@app.post("/items/")
def create_item(item: Item):
    return item


---

📥 Example JSON Input

{
  "name": "Laptop",
  "price": 50000,
  "is_available": true
}


---

🔍 Data Validation (Automatic 🚀)

class Item(BaseModel):
    name: str
    price: float
    quantity: int

❌ Wrong types → automatic error


---

🎯 Field Validation

from pydantic import Field

class Item(BaseModel):
    name: str = Field(..., min_length=3, max_length=50)
    price: float = Field(gt=0)
    quantity: int = Field(ge=1)


---

🧠 Optional Fields

from typing import Optional

class Item(BaseModel):
    name: str
    description: Optional[str] = None


---

🔁 Nested Models

class Manufacturer(BaseModel):
    name: str
    country: str

class Item(BaseModel):
    name: str
    manufacturer: Manufacturer


---

📦 List in Request Body

from typing import List

class Item(BaseModel):
    name: str

@app.post("/items/")
def create_items(items: List[Item]):
    return items


---

🎯 Combining Path + Query + Body

@app.put("/items/{item_id}")
def update_item(
    item_id: int,
    q: str,
    item: Item
):
    return {
        "item_id": item_id,
        "query": q,
        "item": item
    }


---

🔥 Advanced Concepts (Important for Interviews)

✅ Body with Multiple Models

class User(BaseModel):
    username: str

class Item(BaseModel):
    name: str

@app.post("/order/")
def create_order(user: User, item: Item):
    return {"user": user, "item": item}


---

✅ Embed Body

from fastapi import Body

@app.post("/items/")
def create_item(item: Item = Body(embed=True)):
    return item


---

✅ Default Values

class Item(BaseModel):
    name: str
    price: float = 100.0


---

⚠️ Common Mistakes

❌ Forgetting type hints
❌ Not using Pydantic models
❌ Mixing query and body incorrectly
❌ Not validating inputs


---

🧠 Best Practices

✔ Use Pydantic models for all request bodies
✔ Keep endpoints clean and minimal
✔ Use validation (Field, Query, Path)
✔ Follow REST naming conventions
✔ Use Swagger (/docs) for testing
