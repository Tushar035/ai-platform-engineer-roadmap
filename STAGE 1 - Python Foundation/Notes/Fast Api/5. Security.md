# 📌 1. Authentication with OAuth2

## 🧠 What is OAuth2?

OAuth2 is a **standard protocol for authorization**.

👉 It allows:
- Users to log in securely
- Apps to access resources without exposing passwords

---

## 🔐 OAuth2 Flow (Password Flow)

FastAPI supports OAuth2 using `OAuth2PasswordBearer`.

👉 Flow:
1. User sends username + password
2. Server verifies credentials
3. Server returns access token
4. Client uses token for future requests

---

## ✅ Basic OAuth2 Setup

```python
from fastapi import Depends, FastAPI
from fastapi.security import OAuth2PasswordBearer

app = FastAPI()

oauth2_scheme = OAuth2PasswordBearer(tokenUrl="token")

@app.get("/secure")
def secure_route(token: str = Depends(oauth2_scheme)):
    return {"token": token}


---

💡 Explanation

OAuth2PasswordBearer extracts token from request header

Header format:

Authorization: Bearer <token>


👉 FastAPI automatically:

Reads header

Passes token to your function

📌 2. JWT Authentication in FastAPI

🧠 What is JWT?

JWT (JSON Web Token) is a secure token format.

It contains:

Header

Payload (user data)

Signature (security)



---

📦 Install Dependencies

pip install python-jose passlib[bcrypt]


---

🔐 Create JWT Token

from jose import JWTError, jwt
from datetime import datetime, timedelta

SECRET_KEY = "your-secret"
ALGORITHM = "HS256"

def create_access_token(data: dict):
    to_encode = data.copy()
    expire = datetime.utcnow() + timedelta(minutes=30)
    to_encode.update({"exp": expire})
    return jwt.encode(to_encode, SECRET_KEY, algorithm=ALGORITHM)


---

💡 Explanation

exp → token expiry

SECRET_KEY → used to sign token

ALGORITHM → encryption method



---

🔓 Decode JWT Token

def verify_token(token: str):
    try:
        payload = jwt.decode(token, SECRET_KEY, algorithms=[ALGORITHM])
        return payload
    except JWTError:
        raise Exception("Invalid Token")


---

🔐 Login Endpoint (Generate Token)

from fastapi import HTTPException

fake_user = {
    "username": "admin",
    "password": "1234"
}

@app.post("/token")
def login(username: str, password: str):
    if username != fake_user["username"] or password != fake_user["password"]:
        raise HTTPException(status_code=401, detail="Invalid credentials")

    access_token = create_access_token({"sub": username})
    return {"access_token": access_token, "token_type": "bearer"}


---

🔒 Protected Route

from fastapi import Depends

@app.get("/profile")
def profile(token: str = Depends(oauth2_scheme)):
    payload = verify_token(token)
    return {"user": payload}


---

💡 Explanation

Login generates token

Client sends token in header

Server verifies token before allowing access

📌 3. Password Hashing & Security Best Practices

🧠 Why Hash Passwords?

👉 Never store plain passwords.

Hashing:

Converts password → unreadable string

Even DB leak = passwords safe



---

🔐 Password Hashing

from passlib.context import CryptContext

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")

def hash_password(password: str):
    return pwd_context.hash(password)

def verify_password(plain: str, hashed: str):
    return pwd_context.verify(plain, hashed)


---

💡 Explanation

bcrypt → strong hashing algorithm

One-way → cannot reverse



---

🔄 Example Usage

hashed = hash_password("mypassword")

verify_password("mypassword", hashed)  # True


---

⚠️ Best Practices

Use strong hashing (bcrypt/argon2)

Never store raw passwords

Use salt (bcrypt handles automatically)

---

🔐 Token Expiry & Refresh Tokens

💡 Why?

Access tokens expire quickly (security)

Refresh tokens generate new tokens



---

🔄 Example Concept

def create_refresh_token(data: dict):
    expire = datetime.utcnow() + timedelta(days=7)
    return jwt.encode({**data, "exp": expire}, SECRET_KEY, algorithm=ALGORITHM)


---

🔒 Role-Based Access Control (RBAC)

def get_current_user(token: str = Depends(oauth2_scheme)):
    payload = verify_token(token)
    return payload

def admin_only(user=Depends(get_current_user)):
    if user.get("role") != "admin":
        raise Exception("Access denied")
    return user

@app.get("/admin")
def admin_dashboard(user=Depends(admin_only)):
    return {"message": "Welcome Admin"}


---

💡 Explanation

Adds authorization layer

Controls access based on roles



---

🔐 Secure Headers

from fastapi import Response

@app.get("/")
def secure_headers(response: Response):
    response.headers["X-Content-Type-Options"] = "nosniff"
    response.headers["X-Frame-Options"] = "DENY"
    return {"message": "Secure"}
