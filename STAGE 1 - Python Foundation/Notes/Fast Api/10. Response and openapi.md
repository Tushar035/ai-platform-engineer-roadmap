# 📌 1. Response Models & Return Types

---

## 🧠 What is a Response Model?

A **response model** defines what your API returns.

👉 Why important:
- Data validation
- Security (hide fields)
- Documentation (Swagger)

---

## ✅ Basic Example

```python
from pydantic import BaseModel

class UserResponse(BaseModel):
    name: str
    email: str

@app.get("/user/", response_model=UserResponse)
def get_user():
    return {
        "name": "Tushar",
        "email": "tushar@mail.com",
        "password": "secret"  # will be hidden
    }
```

---

💡 Explanation

Only fields in UserResponse are returned

Extra fields are automatically filtered



---

🔄 Response Model with List

from typing import List

@app.get("/users/", response_model=List[UserResponse])
def get_users():
    return [{"name": "A", "email": "a@mail.com"}]


---

🔁 Response Model Exclude Fields

@app.get("/user/", response_model=UserResponse, response_model_exclude={"email"})
def get_user():
    return {"name": "Tushar", "email": "t@mail.com"}


---

🔀 Union Response

from typing import Union

@app.get("/data/", response_model=Union[UserResponse, dict])
def get_data(flag: bool):
    if flag:
        return {"name": "Tushar", "email": "t@mail.com"}
    return {"message": "No data"}

📌 2. Response Status Codes


---

🧠 What are Status Codes?

They indicate:

Success

Failure

Errors



---

✅ Basic Status Code

from fastapi import status

@app.post("/create/", status_code=status.HTTP_201_CREATED)
def create():
    return {"msg": "Created"}


---

🔥 Common Status Codes

Code	Meaning

200	OK
201	Created
400	Bad Request
401	Unauthorized
404	Not Found
500	Server Error



---

❌ Raise Errors

from fastapi import HTTPException

@app.get("/user/{id}")
def get_user(id: int):
    if id != 1:
        raise HTTPException(status_code=404, detail="User not found")
    return {"id": id}


---

💡 Explanation

Stops execution

Returns structured error response

📌 3. Response Headers & Cookies


---

🧠 Why Use Headers?

Metadata

Security info

Rate limits



---

✅ Add Headers

from fastapi import Response

@app.get("/")
def add_header(response: Response):
    response.headers["X-App-Version"] = "1.0"
    return {"msg": "Hello"}


---

🍪 Set Cookies

@app.get("/login/")
def login(response: Response):
    response.set_cookie(key="session_id", value="abc123")
    return {"msg": "Logged in"}


---

❌ Delete Cookie

@app.get("/logout/")
def logout(response: Response):
    response.delete_cookie("session_id")
    return {"msg": "Logged out"}


---

🔐 Secure Cookies

response.set_cookie(
    key="session_id",
    value="abc123",
    httponly=True,
    secure=True
)


---

💡 Explanation

httponly → prevents JS access

secure → HTTPS only

📌 4. Additional Responses in OpenAPI


---

🧠 What is OpenAPI?

FastAPI auto-generates API docs:

/docs (Swagger)

/redoc



---

✅ Add Multiple Responses

@app.get(
    "/items/{id}",
    responses={
        200: {"description": "Success"},
        404: {"description": "Not found"},
    }
)
def get_item(id: int):
    return {"id": id}


---

🔄 Response with Model

@app.get(
    "/items/{id}",
    responses={
        404: {"model": UserResponse, "description": "Error response"}
    }
)
def get_item(id: int):
    return {"id": id}


---

💡 Explanation

Enhances API documentation

Useful for frontend developers

📌 5. Custom Responses (HTML, Streaming, File)


---

🧾 HTML Response

from fastapi.responses import HTMLResponse

@app.get("/", response_class=HTMLResponse)
def html():
    return "<h1>Hello World</h1>"


---

📄 Plain Text Response

from fastapi.responses import PlainTextResponse

@app.get("/", response_class=PlainTextResponse)
def text():
    return "Hello"


---

📁 File Response

from fastapi.responses import FileResponse

@app.get("/file/")
def file():
    return FileResponse("test.pdf")


---

📥 Streaming Response

from fastapi.responses import StreamingResponse

def generate():
    for i in range(5):
        yield f"Chunk {i}\n"

@app.get("/stream/")
def stream():
    return StreamingResponse(generate(), media_type="text/plain")


---

💡 Explanation

Streaming → sends data in chunks

Useful for large data/files



---

🔄 JSONResponse (Custom)

from fastapi.responses import JSONResponse

@app.get("/")
def custom_json():
    return JSONResponse(content={"msg": "Hello"}, status_code=202)


---

🔥 Real-World Example (Combined)

@app.post("/users/", response_model=UserResponse, status_code=201)
def create_user(response: Response):
    response.set_cookie("session_id", "abc")
    return {"name": "Tushar", "email": "t@mail.com"}


---

🔥 Common Mistakes

Returning sensitive data

Not using response_model

Ignoring status codes

Not documenting responses



---

🧠 Best Practices

Always use response_model

Use proper status codes

Secure cookies

Document all responses

Use streaming for large data
