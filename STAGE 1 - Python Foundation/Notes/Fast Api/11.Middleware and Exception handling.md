# 📌 1. Middleware Explained

---

## 🧠 What is Middleware?

Middleware is code that runs:
👉 Before request reaches endpoint  
👉 After response is generated  

Flow:

Request → Middleware → Route → Middleware → Response

---

## ✅ Basic Middleware Example

```python
from fastapi import FastAPI, Request

app = FastAPI()

@app.middleware("http")
async def log_middleware(request: Request, call_next):
    print("Before request")

    response = await call_next(request)

    print("After response")

    return response
```

---

💡 Explanation

request → incoming request

call_next() → passes request to next handler

You can modify request/response



---

⏱️ Measure Response Time

import time

@app.middleware("http")
async def add_process_time(request: Request, call_next):
    start = time.time()

    response = await call_next(request)

    process_time = time.time() - start
    response.headers["X-Process-Time"] = str(process_time)

    return response


---

💡 Real-world use:

Performance monitoring

API latency tracking

📌 2. Advanced Middleware Patterns


---

📊 Logging Middleware

@app.middleware("http")
async def logging_middleware(request: Request, call_next):
    print(f"Request: {request.method} {request.url}")

    response = await call_next(request)

    print(f"Response status: {response.status_code}")
    return response


---

💡 Explanation

Logs every request

Useful for debugging and monitoring



---

🔥 Structured Logging (Production)

import logging

logger = logging.getLogger("app")

@app.middleware("http")
async def structured_logging(request: Request, call_next):
    logger.info(f"{request.method} {request.url}")

    response = await call_next(request)

    logger.info(f"Status: {response.status_code}")
    return response


---

🚨 Exception Handling Middleware

from fastapi.responses import JSONResponse

@app.middleware("http")
async def error_handler(request: Request, call_next):
    try:
        return await call_next(request)
    except Exception as e:
        return JSONResponse(
            status_code=500,
            content={"error": str(e)}
        )


---

💡 Explanation

Catches unhandled errors

Prevents app crash

Returns consistent error response



---

🔐 Security Middleware (Example)

@app.middleware("http")
async def security_headers(request: Request, call_next):
    response = await call_next(request)
    response.headers["X-Frame-Options"] = "DENY"
    response.headers["X-Content-Type-Options"] = "nosniff"
    return response

📌 3. Lifespan Events (Startup & Shutdown)


---

🧠 What are Lifespan Events?

Run code when:

App starts 🚀

App stops 🛑



---

✅ Startup Event

@app.on_event("startup")
async def startup():
    print("App started")


---

🛑 Shutdown Event

@app.on_event("shutdown")
async def shutdown():
    print("App stopped")


---

💡 Explanation

Used for:

DB connection setup

Cache initialization

Resource cleanup



---

🔥 Lifespan Context (Modern Way)

from contextlib import asynccontextmanager

@asynccontextmanager
async def lifespan(app: FastAPI):
    print("Startup logic")
    yield
    print("Shutdown logic")

app = FastAPI(lifespan=lifespan)


---

💡 Why Use This?

Cleaner structure

Better for complex apps

📌 4. Using Request Object Directly


---

🧠 What is Request Object?

Gives full access to request:

Headers

Body

Query params

Client info



---

✅ Basic Example

from fastapi import Request

@app.get("/")
async def read_request(request: Request):
    return {
        "method": request.method,
        "url": str(request.url)
    }


---

📦 Access Headers

@app.get("/headers/")
async def headers(request: Request):
    return dict(request.headers)


---

📥 Read Body

@app.post("/body/")
async def read_body(request: Request):
    body = await request.json()
    return body


---

🌐 Client Info

@app.get("/client/")
async def client(request: Request):
    return {"client": request.client.host}


---

💡 Explanation

Useful for:

Logging

Custom parsing

Middleware logic




---

🔥 Real-World Example (Combined)

@app.middleware("http")
async def full_middleware(request: Request, call_next):
    print(f"{request.method} {request.url}")

    response = await call_next(request)

    response.headers["X-App"] = "FastAPI"
    return response
