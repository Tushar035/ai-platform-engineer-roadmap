# 📌 1. Async Programming in FastAPI

---

## 🧠 What is Async Programming?

Async = Non-blocking execution

👉 Instead of waiting for one task to finish:
- Multiple tasks run concurrently

---

## 🔄 Sync vs Async

### ❌ Sync (Blocking)

```python
def get_data():
    data = slow_db_call()
    return data

👉 Problem:

Blocks the server

Poor performance



---

✅ Async (Non-Blocking)

async def get_data():
    data = await slow_db_call()
    return data

👉 Benefits:

Handles many requests simultaneously

Better performance



---

⚙️ How FastAPI Uses Async

FastAPI is built on:

ASGI (Asynchronous Server Gateway Interface)

Starlette

Uvicorn



---

✅ Async Route Example

from fastapi import FastAPI
import asyncio

app = FastAPI()

@app.get("/")
async def home():
    await asyncio.sleep(1)
    return {"message": "Async response"}


---

💡 Explanation

async def → defines async function

await → waits without blocking



---

🔁 When to Use Async?

Use async when:

Calling APIs

DB operations (async drivers)

File I/O

Network calls



---

⚠️ When NOT to Use Async?

❌ CPU-heavy tasks:

Image processing

Machine learning


👉 Use:

Celery

Background workers



---

🔄 Mixing Sync & Async

@app.get("/sync")
def sync_route():
    return {"msg": "sync"}

@app.get("/async")
async def async_route():
    return {"msg": "async"}

👉 FastAPI handles both automatically


---

🔥 Async Database Example

from sqlalchemy.ext.asyncio import create_async_engine, AsyncSession
from sqlalchemy.orm import sessionmaker

DATABASE_URL = "postgresql+asyncpg://user:password@localhost/db"

engine = create_async_engine(DATABASE_URL)

AsyncSessionLocal = sessionmaker(engine, class_=AsyncSession)

async def get_db():
    async with AsyncSessionLocal() as session:
        yield session


---

💡 Explanation

asyncpg → async PostgreSQL driver

AsyncSession → non-blocking DB operations

📌 2. Background Tasks


---

🧠 What are Background Tasks?

Tasks that run after response is sent

👉 Examples:

Send email

Log activity

Process file



---

✅ Basic Example

from fastapi import BackgroundTasks

def write_log(message: str):
    with open("log.txt", "a") as f:
        f.write(message)

@app.get("/log")
def log(background_tasks: BackgroundTasks):
    background_tasks.add_task(write_log, "User accessed API\n")
    return {"message": "Logged"}


---

💡 Explanation

Runs after response

Improves API speed



---

🔄 Multiple Background Tasks

def task1():
    print("Task 1")

def task2():
    print("Task 2")

@app.get("/")
def run_tasks(bg: BackgroundTasks):
    bg.add_task(task1)
    bg.add_task(task2)
    return {"msg": "Tasks running"}


---

⚡ Async Background Task

async def async_task():
    await asyncio.sleep(2)
    print("Done")

@app.get("/")
async def run():
    asyncio.create_task(async_task())
    return {"msg": "Started"}


---

⚠️ Limitations

Runs in same process

Not suitable for heavy workloads


👉 Use:

Celery

Redis Queue



---

🧠 Real-World Example

def send_email(email: str):
    print(f"Sending email to {email}")

@app.post("/register")
def register(email: str, bg: BackgroundTasks):
    bg.add_task(send_email, email)
    return {"msg": "User created"}

📌 3. Async Testing


---

🧠 Why Async Testing?

Ensure:

Async endpoints work correctly

No hidden bugs



---

📦 Install

pip install pytest httpx pytest-asyncio


---

✅ Basic Test Example

import pytest
from httpx import AsyncClient
from main import app

@pytest.mark.asyncio
async def test_home():
    async with AsyncClient(app=app, base_url="http://test") as ac:
        response = await ac.get("/")
    
    assert response.status_code == 200


---

💡 Explanation

AsyncClient → async HTTP client

pytest.mark.asyncio → async test support



---

🔄 Testing POST API

@pytest.mark.asyncio
async def test_create_user():
    async with AsyncClient(app=app, base_url="http://test") as ac:
        response = await ac.post("/users/", json={"name": "Tushar"})
    
    assert response.status_code == 200


---

🧪 Testing with DB

@pytest.mark.asyncio
async def test_db():
    async with AsyncClient(app=app, base_url="http://test") as ac:
        response = await ac.get("/users/1")
    
    assert response.json()["id"] == 1

🔥 Mocking Dependencies

from fastapi.testclient import TestClient

def override_get_db():
    return "test_db"

app.dependency_overrides[get_db] = override_get_db


---

💡 Explanation

Replace real DB with mock

Faster and safer testing



---

🔥 Common Mistakes

Using async without await

Blocking calls inside async

Using sync DB in async routes

Heavy tasks in BackgroundTasks



---

🧠 Best Practices

Use async for I/O operations

Use BackgroundTasks for lightweight work

Use Celery for heavy jobs

Write async tests

Avoid blocking code
