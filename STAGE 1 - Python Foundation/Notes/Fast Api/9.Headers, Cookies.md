# 📌 1. Headers & Cookie Parameters

---

## 🧠 What are Headers?

Headers are metadata sent with requests.

👉 Common examples:
- Authorization token
- Content-Type
- Custom metadata

---

## ✅ Reading Headers

```python
from fastapi import Header

@app.get("/headers/")
def read_headers(user_agent: str = Header()):
    return {"User-Agent": user_agent}
```

---

💡 Explanation

Header() extracts value from request headers

Automatically maps header name → Python variable


👉 Example:

User-Agent: Mozilla/5.0


---

🔄 Optional Headers

from typing import Optional

@app.get("/headers/")
def read_headers(token: Optional[str] = Header(None)):
    return {"token": token}


---

🎯 Custom Header Name

@app.get("/headers/")
def custom_header(x_token: str = Header()):
    return {"X-Token": x_token}

👉 Header:

X-Token: abc123


---

⚠️ Convert Underscores to Hyphens

@app.get("/headers/")
def read_header(user_agent: str = Header(convert_underscores=True)):
    return {"User-Agent": user_agent}


---

🍪 What are Cookies?

Cookies store small data in browser.

👉 Used for:

Sessions

Tracking

Preferences



---

✅ Reading Cookies

from fastapi import Cookie

@app.get("/cookies/")
def read_cookie(session_id: str = Cookie()):
    return {"session_id": session_id}


---

🔄 Optional Cookies

@app.get("/cookies/")
def read_cookie(session_id: str = Cookie(None)):
    return {"session_id": session_id}


---

🍪 Setting Cookies

from fastapi import Response

@app.get("/set-cookie/")
def set_cookie(response: Response):
    response.set_cookie(key="session_id", value="abc123")
    return {"message": "Cookie set"}


---

💡 Explanation

Server sends cookie in response

Browser stores and sends it back automatically

📌 2. Cookie/Header Models


---

🧠 Why Models?

👉 Instead of many parameters → use structured models


---

✅ Header Model

from pydantic import BaseModel

class HeaderModel(BaseModel):
    x_token: str
    user_agent: str

@app.get("/headers/")
def get_headers(headers: HeaderModel = Header()):
    return headers


---

🍪 Cookie Model

class CookieModel(BaseModel):
    session_id: str

@app.get("/cookies/")
def get_cookies(cookies: CookieModel = Cookie()):
    return cookies


---

💡 Explanation

Groups related fields

Improves readability

Better validation

📌 3. Forms & Form Models


---

🧠 What are Forms?

Forms send data as:

application/x-www-form-urlencoded

👉 Used in:

Login forms

HTML submissions



---

📦 Install Dependency

pip install python-multipart


---

✅ Basic Form Example

from fastapi import Form

@app.post("/login/")
def login(username: str = Form(), password: str = Form()):
    return {"username": username}


---

💡 Explanation

Form() extracts form data

Required for HTML forms



---

🔐 OAuth2 Login Example

from fastapi.security import OAuth2PasswordRequestForm

@app.post("/login/")
def login(form_data: OAuth2PasswordRequestForm = Depends()):
    return {"username": form_data.username}


---

🧱 Form Models

class LoginForm(BaseModel):
    username: str
    password: str

@app.post("/login/")
def login(form: LoginForm):
    return form


---

⚠️ Important

Pydantic models don’t directly support Form()
👉 Use dependency injection for advanced cases

📌 4. Request Forms & File Uploads


---

📁 Upload File

from fastapi import File, UploadFile

@app.post("/upload/")
def upload(file: UploadFile = File()):
    return {"filename": file.filename}


---

💡 Explanation

UploadFile → efficient file handling

Uses temporary file storage



---

📦 Multiple Files

from typing import List

@app.post("/upload/")
def upload(files: List[UploadFile] = File()):
    return {"files": [f.filename for f in files]}


---

📄 File Content

@app.post("/upload/")
async def read_file(file: UploadFile):
    content = await file.read()
    return {"size": len(content)}


---

🔄 Save File

@app.post("/upload/")
async def save_file(file: UploadFile):
    with open(file.filename, "wb") as f:
        content = await file.read()
        f.write(content)
    return {"msg": "Saved"}


---

📥 Form + File Together

@app.post("/upload/")
async def upload(
    file: UploadFile = File(),
    description: str = Form()
):
    return {
        "filename": file.filename,
        "description": description
    }


---

⚠️ File Upload Limits

👉 Important for production:

Limit file size

Validate file type



---

🧠 Real-World Example

@app.post("/profile/")
async def upload_profile(
    name: str = Form(),
    image: UploadFile = File()
):
    return {"name": name, "image": image.filename}


---

🔥 Common Mistakes

Forgetting python-multipart

Using JSON instead of Form

Not validating file size/type

Not securing cookies



---

🧠 Best Practices

Use headers for auth tokens

Use cookies for sessions

Validate all inputs

Limit file uploads

Use HTTPS for secure cookies
