🧩 Prompt Engineering (Very Important)


---

📌 What is a Prompt?

Prompt = Input text you give to model



---

🧾 Example Prompt:

> "Where is Ganymede located?"




---

🧠 How Model Responds

Model:

Predicts next words

Generates answer




---

🔄 Prompt → Output Flow

User Input (Prompt)
        ↓
Model Processing
        ↓
Next Word Prediction
        ↓
Generated Output


---

📦 Key Terms Explained


---

1. Prompt

Input text given to model



---

2. Completion

Output generated by model


👉 Includes:

Original prompt + generated answer



---

3. Inference

Process of generating output



---

4. Context Window

Memory space for prompt

Typically:

Few thousand words




---

🧠 Example:

If context window = 4000 words

Model only remembers:

Last 4000 words




---

🧪 Example Explained (From Transcript)


---

🧾 Prompt:

> "Where is Ganymede located in the solar system?"




---

🔄 Process:

Prompt
   ↓
Model Reads Question
   ↓
Understands Context
   ↓
Predicts Answer
   ↓
Generates Completion


---

✅ Output:

Ganymede is:

A moon of Jupiter

Located in Jupiter’s orbit




---

🧠 Why Model Answered Correctly?

Learned from:

Massive training data


Recognized:

"Ganymede" + "solar system"
